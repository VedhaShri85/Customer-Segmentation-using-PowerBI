# Customer-Segmentation-using-PowerBI
A comprehensive Power BI solution for visualizing customer behavior, demographics, and engagement. Features a robust data model, custom themes, and dynamic reporting layouts to drive data-driven decision-making.

## Project Overview
This repository contains the customer.pbix Power BI project, designed to provide a 360-degree view of customer data. The project leverages Power BI's latest features (including the CY24SU10 Base Theme) to deliver high-performance analytics and modern data visualizations.

## Architecture
The project is structured according to standard Power BI development practices:
Data Model: A structured relational model defined in the DataModel layer to support complex DAX measures.
Theme & UI: Utilizes the CY24SU10 JSON theme for consistent branding, color palettes, and typography.
Security: Includes SecurityBindings to manage data access and potentially Row-Level Security (RLS).
Dynamic Layout: Optimized Report/Layout configurations for both desktop and mobile viewing.

## Key Features
Interactive Visualizations: Deep-dive into customer segments and performance metrics.
Automated Data Refresh: Designed for seamless integration with cloud or on-premise data sources.
Custom Metadata: Includes rich metadata tagging for better discoverability and governance.
